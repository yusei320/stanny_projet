## 🚀 Installation (3 minutes)

```bash
# 1. Extraire
unzip techservice-FINAL-COMPLET.zip
cd techservice-FINAL

# 2. Installer
cd backend
npm install

# 3. Lancer
npm start
```

Puis ouvrir : **http://localhost:3000/auth/login.html**
